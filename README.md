# 介紹
- 看影片實作的練習項目
- 使用SpringBoot開發的員工管理系統
- 影片地址：https://www.bilibili.com/video/BV1PE411i7CV?p=1
- 使用偽造數據庫

# 功能

- 員工的CRUD
- 登陸檢查（需任意用戶名+固定密碼：123456才能登陸成功）
- 攔截器
- 國際化

# 使用方式
- 編譯運行後本地瀏覽器訪問 localhost:8080
- 輸入任意用戶名 + 固定密碼123456即可成功進入員工管理介面
- 點擊左側【員工管理】
- 進行員工信息的增刪改查
